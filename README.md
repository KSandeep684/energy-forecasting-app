# energy-forecasting-app

This project is an interactive energy consumption forecasting dashboard built using Streamlit.
It uses LSTM (Long Short-Term Memory) deep learning models to predict future energy usage based on historical AEP hourly data.

The app includes:

 Dataset upload
 Statistical summary
 Visual data exploration
 LSTM model training
 Forecasting next 100 days
 Visualization of actual vs predicted values
 Export predictions as CSV
